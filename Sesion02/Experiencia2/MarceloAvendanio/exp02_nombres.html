Introducción

En esta página se aplican estilos únicamente usando el tipo de cada elemento. No se emplean clases ni identificadores en el CSS para demostrar el alcance de los selectores de nombre.

Esto ayuda a comprender la cascada básica y cómo los estilos generales afectan a toda la estructura del documento.

Conclusiones

  1. Los selectores por nombre aplican estilos amplios y globales.
  2. Limitan la especificidad y pueden requerir estructura clara para no sobre-estilizar.
  3. Son útiles para establecer una base inicial (reset o estilos genéricos).
