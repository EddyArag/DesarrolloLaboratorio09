Mi experiencia en el laboratorio

Introducción

En esta sesión del laboratorio aprendí a crear archivos HTML y CSS, enlazarlos correctamente y estructurar una página web básica. Me resultó interesante experimentar con los estilos y ver cómo los cambios en el CSS afectan la presentación del contenido.

Objetivos

  • Comprender la estructura básica de un documento HTML.
  • Aprender a enlazar archivos CSS externos.
  • Aplicar estilos para mejorar la presentación de una página web.

Desarrollo

Durante la práctica, creé un archivo HTML con etiquetas semánticas y lo vinculé a una hoja de estilos CSS. Modifiqué colores, fuentes y márgenes para observar los efectos en la visualización. Además, utilicé selectores de clase y de etiqueta para personalizar diferentes partes del contenido.

Conclusiones

La experiencia fue enriquecedora, ya que me permitió afianzar los conceptos básicos de desarrollo web y comprender la importancia de separar el contenido de la presentación mediante el uso de CSS.
