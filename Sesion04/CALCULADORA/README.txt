// README.txt
// Proyecto: Calculadora Científica Básica
// Todos los nombres de clases, IDs y variables están en español y comentados.

Este proyecto está diseñado para ser expuesto por 4 personas. Cada archivo contiene comentarios para facilitar la explicación.

1. calculadora.html
   - Estructura HTML extensa, clases e IDs en español y descriptivos.
   - Comentarios en cada sección para facilitar la comprensión.

2. calculadora.css
   - Estilos modernos, grid, responsive.
   - Selectores y comentarios en español.
   - Explica cómo se adapta a móviles y la personalización de botones.

3. calculadora.js
   - Lógica de operaciones básicas.
   - Variables, funciones y comentarios en español.
   - Explica el manejo de eventos, funciones y control de errores.

4. calculadora-cientifica.js
   - Lógica de operaciones científicas y manejo de excepciones.
   - Variables, funciones y comentarios en español.
   - Historial de operaciones científicas.

5. README.txt
   - Esta guía para la exposición.

Sugerencia para exposición:
- Persona 1: HTML y estructura.
- Persona 2: CSS, grid y responsive.
- Persona 3: JS básico (operaciones, display).
- Persona 4: JS científico, historial y eventos.

Todos los archivos están comentados para facilitar la comprensión.
