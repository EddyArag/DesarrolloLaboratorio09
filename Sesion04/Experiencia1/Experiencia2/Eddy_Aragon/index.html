Experiencia de Práctica N° 02: Uso de Recursos del Lenguaje

a) Uso de Variables y Constantes

Se crea un bloque de código donde se declara una variable y una constante con ámbito de bloque. Se demuestra su alcance utilizando console.log(). 


{
    let variableDeBloque = "Soy una variable de bloque";
    const constanteDeBloque = "Soy una constante de bloque";
    console.log(variableDeBloque); // Accesible aquí
    console.log(constanteDeBloque); // Accesible aquí
}
// console.log(variableDeBloque); // Error: no accesible fuera del bloque
// console.log(constanteDeBloque); // Error: no accesible fuera del bloque

Las variables y constantes declaradas con let y const solo existen dentro del bloque donde se declaran.

b) Elevación de Variables (Hoisting)


console.log(x === undefined); // true
var x = 3;
// devolverá un valor de undefined
var myvar = 'my value';
(function() {
    console.log(myvar); // undefined
    var myvar = 'valor local';
})();

Descripción: Las declaraciones con var se elevan (hoisting) al inicio del contexto, pero no su inicialización. Por eso, x y myvar existen pero son undefined antes de su asignación.

c) Uso de console.log y Scope


let sinAsignar;
console.log(sinAsignar); // undefined

function externa() {
    let a = 1;
    function interna() {
        let b = 2;
        console.log(a); // 1
        console.log(b); // 2
    }
    interna();
    // console.log(b); // Error: b no está definida aquí
}
externa();

Descripción: console.log() permite verificar el valor y el alcance de las variables en diferentes contextos.

d) Tipos de Datos y Cambio de Tipos


var answer = 42;
answer = 'Gracias por todo el pescado...';

Descripción: En JavaScript, las variables pueden cambiar de tipo dinámicamente. Aquí, answer pasa de ser un número a una cadena. 


parseInt("F", 16);      // 15
parseInt("17", 8);      // 15
parseInt("15", 10);     // 15
parseInt(15.99, 10);    // 15
parseInt("FXX123", 16); // 15
parseInt("1111", 2);    // 15
parseInt("15*3", 10);   // 15
parseInt("12", 13);     // 15
parseInt("Hello", 8);   // NaN
parseInt("0x7", 10);    // 0
parseInt("546", 2);     // NaN

Descripción: parseInt() convierte cadenas a números según la base indicada. Si la cadena no es válida para la base, devuelve NaN. 


var howMany = 10;
alert("howMany.toString() is " + howMany.toString()); // displays "10"
alert("45 .toString() is " + 45 .toString()); // displays "45"
var x = 7;
alert(x.toString(2)); // Displays "111"

Descripción: toString() convierte un número a cadena. Puede recibir una base para mostrar el número en binario, octal, etc.
